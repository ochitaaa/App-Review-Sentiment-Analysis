# App-Review-Sentiment-Analysis

This project focuses on sentiment analysis of application reviews collected through web scraping. The scraped data is processed and analyzed using machine learning techniques to classify user sentiments. This project was developed as a module submission for the Asah by Dicoding program.

---

## Project Objectives
- Collect application review data through web scraping  
- Perform data cleaning and preprocessing on textual data  
- Build and train sentiment analysis models  
- Evaluate model performance for sentiment classification  

---

## Methods & Models Used
- Text preprocessing and cleaning  
- Feature extraction for text data  
- Machine learning-based sentiment classification  

---

## Repository Structure

- `Data_Scraping_Rosita_Angel_Cahyadi.ipynb`  
  Notebook for scraping application review data.

- `Pelatihan_Model_Analisis_Sentimen_Rosita_Angel_Cahyadi.ipynb`  
  Notebook for sentiment analysis model training and evaluation.

- `ulasan_aplikasi.csv`  
  Raw application review data obtained from web scraping.

- `cleaned_reviews.csv`  
  Cleaned and preprocessed review dataset used for modeling.

- `requirements.txt`  
  List of Python dependencies required to run this project.
